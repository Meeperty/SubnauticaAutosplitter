# SubnauticaAutosplitter
Source control for subnautica autosplitter development.  
For help using it, ask in the [Subnautica Speedrun Discord in #help](https://discord.gg/vgEC5D4jkM)

The autosplitter currently supports:  
  * Starting (on skipping the start cutscene)  
  * Ending (on rocket launch)  
  * Splitting (on QEP gun deactivation)  

On these patches:  
  * September 2018 (aka old patch)  
  * December 2021 (aka legacy patch)  
  * March 2023 (the most recent patch)
